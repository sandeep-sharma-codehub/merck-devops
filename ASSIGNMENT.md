Objective: 
The goal of this challenge is to assess your proficiency in cloud services (specifically AWS), and 
your ability to design and implement a reliable and scalable infrastructure. You will demonstrate your 
skills in automation, configuration management, and continuous deployment. 

Challenge Overview 
You are tasked with creating a simplified cloud-based application deployment pipeline using GitHub 
Actions and AWS services.  
• You will create a basic Python REST API and deploy it to AWS using a CI/CD pipeline.  
• The REST API shall have at least two endpoints, one for checking the application health and one 
to retrieve (dummy) data and shall require authentication.  
• The application should be containerized using Docker. Please use the AWS , and GitHub 
repositories as applicable. 
•  Please use AWS CDK or CloudFormation to define the infrastructure.  
• Write tests where relevant and follow market standard practices. 
• Ensure to document the code and your approach to the solution. 
• make sure application and code is enterprise grade
• full CICD must be operational
